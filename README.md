## 🇺🇸 English Version

### 📖 Overview
The ultimate `SKILL.md` system prompt for **Vibe Coders** using **Google Antigravity**. It transforms your AI agent into a Staff-Level **Autonomous CTO**. This protocol proactively defends your codebase against context degradation, spaghetti code, and architectural failures.

### ✨ Key Features
* **Proactive `<Context_Anchor>`:** Prevents hallucinations by forcing the AI to refresh its project knowledge before writing code.
* **Atomic Execution:** Implements changes step-by-step with a verification loop to prevent "Big Bang" crashes.
* **LEES Matrix:** Mandates resilient Frontend UX (Loading, Error, Empty, Success states).
* **TDC (Test-Driven Criticality):** Mandatory automated tests for all critical business logic.

### 🚀 Installation

#### Option 1: Terminal One-Liner (Fastest)
Run this command in your project's root directory to install the skill instantly:

```bash
mkdir -p .agent/skills/agentic-cto-prime && curl -o .agent/skills/agentic-cto-prime/SKILL.md [https://raw.githubusercontent.com/bmubook/Agentic-CTO-Prime/main/SKILL.md](https://raw.githubusercontent.com/bmubook/Agentic-CTO-Prime/main/SKILL.md)
```

#### Option 2: Manual Setup
1. Create the folder: `.agent/skills/agentic-cto-prime/`
2. Place the `SKILL.md` file from this repo into that folder.

## 🇸🇦 النسخة العربية

### 📖 نظرة عامة
الدستور الهندسي المطلق (Agentic CTO Prime) للمبرمجين المعتمدين على توجيه الذكاء الاصطناعي (Vibe Coders). مصمم للعمل بأعلى كفاءة مع وكيل **Google Antigravity** ليحوله من مجرد "طبّاع أكواد" إلى مدير تقني تنفيذي (CTO) محترف يحمي مشروعك من الانهيارات والهلوسة البرمجية.

### ✨ الميزات الأساسية
* **الإرساء الاستباقي للذاكرة `<Context_Anchor>`:** يمنع الهلوسة عبر إجبار الوكيل على تحديث معلوماته عن المشروع قبل البدء بكتابة أي كود.
* **التنفيذ الذري (Atomic Execution):** تنفيذ التعديلات خطوة بخطوة مع التحقق من كل مرحلة لمنع الانهيارات المفاجئة.
* **مصفوفة LEES للواجهات:** معالجة احترافية وإلزامية لحالات (التحميل، الخطأ، الفراغ، والنجاح) في واجهة المستخدم.
* **الاختبارات الإجبارية (TDC):** إلزام الوكيل بكتابة اختبارات مؤتمتة لأي مسار برمجي حساس (مثل الدفع أو قواعد البيانات).
